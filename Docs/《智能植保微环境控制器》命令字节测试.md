# 《智能植保微环境控制器》命令字节测试

## 1. 基础指令

1. 获取传感器数据：

   ```
   AA 01 00 01 55
   
   响应示例：
   0xAA, 0x09, 0x0D, 0xE8, 0x03, 0xFA, 0x00, 0xEA, 0x01, 0xF3, 0x00, 0x0F, 0xD2, 0x23, 0x1A, 0x00, 0xE9, 0x55
   ```

2. 获取执行器数据：

   ```
   AA 02 00 02 55
   
   响应示例：
   0xAA, 0x09, 0x03, 0x00, 0x00, 0x01, 0x0B, 0x55
   ```

3. 设置执行器：

   1. 开启风扇：

      ```
      AA 03 02 01 01 01 55
      
      响应示例：
      0xAA, 0x0A, 0x00, 0x0A, 0x55
      ```

   2. 关闭风扇：

      ```
      AA 03 02 01 00 00 55
      
      响应示例：
      0xAA, 0x0A, 0x00, 0x0A, 0x55
      ```

4. 设置控制参数：

   ```
   AA 04 20 00 00 20 42 00 00 70 42 00 00 A0 41 00 00 90 41 00 00 FA 43 00 00 48 44 05 00 00 00 14 00 00 00 E0 55
   
   响应示例：
   0xAA, 0x09, 0x00, 0x09, 0x55
   ```
   
5. 获取控制参数：

   ```
   AA 05 00 05 55
   
   响应示例：
   0xAA, 0x09, 0x20, 0x00, 0x00, 0x20, 0x42, 0x00, 0x00, 0x70, 0x42, 0x00, 0x00, 0xF0, 0x41, 0x00, 0x00, 0xA0, 0x41, 0x00, 0x00, 0xFA, 0x43, 0x00, 0x00, 0x48, 0x44, 0x05, 0x00, 0x00, 0x00, 0x14, 0x00, 0x00, 0x00, 0x8D, 0x55
   ```

6. 复位系统：

   ```
   AA 06 00 06 55
   
   响应示例：
   0xAA, 0x09, 0x00, 0x09, 0x55
   ```

7. 获取系统信息：

   ```
   AA 08 00 08 55
   
   响应示例：
   0xAA, 0x09, 0x0C, 0x01, 0x00, 0x00, 0x00, 0xCD, 0x1F, 0x1B, 0x00, 0x00, 0x00, 0x00, 0x20, 0xED, 0x55
   ```

8. 设置控制模式：

   1. 手动模式：

      ```
      AA 0B 01 01 0B 55
      
      响应示例：
      0xAA, 0x09, 0x00, 0x09, 0x55
      ```

   2. 自动模式：

      ```
      AA 0B 01 00 0B 55
      
      响应示例：
      0xAA, 0x09, 0x00, 0x09, 0x55
      ```

9. 等等
